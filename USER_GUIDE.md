# User Guide

## Getting Started
1. Open the quiz home page.
2. Click `ENTER PORTAL`.
3. Select `Study Mode` or `Exam Mode`.

## Study Mode
- Choose study type and category.
- Start session and answer questions.
- Review feedback and detailed results.

## Exam Mode
- Choose category and question count.
- Complete timed exam.
- Use review screen before submit.

## Dashboard
- Open `Performance Dashboard` from menu.
- Track total attempts, overall accuracy, and weak areas.

## Tips
- Use weak-area practice for low-accuracy categories.
- Finish sessions to keep progress consistent.
